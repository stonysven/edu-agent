# AI学习建议

学习 AI 可以分成四个阶段：

第一阶段是基础能力建设。建议先学习 Python、线性代数、概率统计和基本算法思维。

第二阶段是机器学习基础。建议理解监督学习、过拟合、损失函数、梯度下降、模型评估等核心概念。

第三阶段是深度学习。可以继续学习神经网络、反向传播、卷积网络、Transformer 和大语言模型基础。

第四阶段是项目实践。通过做小项目把知识串起来，比如做一个问答机器人、文本分类器或简单的 RAG 系统。

在学习顺序上，建议先打基础，再做项目，不要一开始只追模型名词。

如果目标是做 AI Agent 系统，除了模型本身，还要理解：
- Prompt 设计
- Tool 调用
- Memory
- RAG
- 多轮对话状态管理
